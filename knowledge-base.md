# Helpdesk Knowledge Base

This file contains support articles used by the AI to automatically resolve incoming support tickets.

## Refund Policy & Billing Requests
- **Issue:** Customers asking for a refund, billing errors, double charges, or invoice issues.
- **Resolution:** We offer a 100% money-back guarantee for all purchases made within the last 30 days. To request a refund, please specify your invoice number or transaction ID. Refunds are processed back to the original payment method within 5-7 business days.
- **Auto-Resolve Criteria:** If the customer asks for a refund for a purchase within the last 30 days and provides their billing details/email, or if they report a clear double charge, the ticket can be auto-resolved with instructions that a refund has been initiated or the billing department has been notified to process it.

## Password Reset & Login Troubleshooting
- **Issue:** Users unable to log in, forgotten passwords, or password reset links not working.
- **Resolution:** If you've forgotten your password, go to the login page and click "Forgot Password" or visit https://codewithai.com/forgot-password. Enter your registered email address, and a secure password reset link will be sent to you immediately. If the link has expired, please request a new one, as they are only valid for 1 hour.
- **Auto-Resolve Criteria:** If the user is asking how to reset their password or reports that they forgot their password, the ticket can be auto-resolved by providing the link and instructions.

## API Documentation & Webhooks
- **Issue:** Developers asking about API documentation, webhook setup, or rate limits.
- **Resolution:** Our complete API documentation and webhook guides are available at https://codewithai.com/docs/api. The API rate limit is 100 requests per minute for Starter plans and 1000 requests per minute for Enterprise plans. Webhooks can be configured under your Account Settings > Developer Portal.
- **Auto-Resolve Criteria:** If the user asks for API documentation, webhook setup instructions, or API rate limit queries, the ticket can be auto-resolved by pointing them to the developer docs link.

## Pricing & Plan Upgrades
- **Issue:** General inquiries about pricing tiers, annual discounts, or upgrades.
- **Resolution:** Our pricing plans can be viewed at https://codewithai.com/pricing. We offer Starter ($15/mo), Professional ($49/mo), and Enterprise (custom) tiers. We offer a 15% discount for annual billing on all plans. To upgrade your plan, go to your Billing settings in the dashboard.
- **Auto-Resolve Criteria:** If the user inquires about generic pricing options, annual discounts, or tier pricing, the ticket can be auto-resolved by providing our plan pricing list.
